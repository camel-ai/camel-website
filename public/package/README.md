# Branding Package Downloads

Place the following files in this directory for the branding page downloads:

| File                       | Description                   |
| -------------------------- | ----------------------------- |
| `eigent-logo.zip`          | Eigent AI logo assets         |
| `palatino.zip`             | Palatino font files           |
| `camel-branding-pack.zip`  | CAMEL AI logos and font       |
| `built-with-camel.zip`     | Built with CAMEL badge assets |
| `camel-paper-template.zip` | CAMEL LaTeX paper template    |

Files are served at `/package/<filename>` in the branding page.
